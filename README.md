📱 FLASHCARD MASTER - Clean GitHub Description

A stunning, AI-inspired flashcard application that combines beautiful data-visualization design with powerful spaced repetition technology.

✨ Features

🎨 Beautiful Design
- Dark theme with vibrant era-based colors (orange, cyan, purple)
- Gradient deck cards with shadows and smooth animations
- 3D flip animations with haptic feedback
- Subtle dot grid background inspired by data visualization

🧠 Smart Learning
- SM-2 spaced repetition algorithm (same as Anki)
- Progress tracking with card states: New, Learning, Young, Mature, Mastered
- Quality-based ratings (0-5 scale) for precise recall tracking

📚 Complete Control
- Multiple decks with custom icons and colors
- Add images from gallery to card fronts and backs
- Create, edit, and delete decks and cards

📱 Interactive Experience
- Shake phone to shuffle cards during review
- Haptic feedback for flips, successes, and errors
- Smooth 3D card flip animations

🔒 Privacy First
- 100% offline - no internet required
- Zero data collection - all data stays on your device
- Open source and transparent

 🛠️ Built With
- Kotlin & Jetpack Compose
- Room Database
- Coroutines & Flow
- Material 3 Design
- Coil for image loading

 🚀 Quick Start
1. Open in Android Studio
2. Sync Gradle
3. Run on device or emulator (API 24+)

Perfect for students, language learners, and professionals who want a beautiful, private, and scientifically-optimized learning tool.
